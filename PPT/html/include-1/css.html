Part2-1CSS

CSSCascading Style Sheets
종속형 시트 또는 캐스케이딩 스타일 시트(영어: Cascading Style Sheet)는 마크업 언어가 실제 표시되는 방법을 기술하는 스타일 언어(영어: Style sheet language 스타일 시트 랭귀지)로, W3C의 표준이고, 레이아웃과 스타일을 정의할 때의 자유도가 높습니다.

마크업 언어(ex: HTML)가 웹사이트의 몸체를 담당한다면 CSS는 옷과 액세서리처럼 꾸미는 역할을 담당한다고 할 수 있습니다. 즉, HTML 구조는 그대로 두고 CSS 파일만 변경해도 전혀 다른 웹사이트처럼 꾸밀 수 있습니다.

Part2-2CSS 사용법

태그 내에 스타일 지정
(Inline Style)

태그에 직접 스타일(CSS)를 적용하는 방법입니다


                <p style="color:blue;">타란유엑스디는 좀 짱인듯?</p>
내부 스타일 시트
(Internal Style Sheet)

HTML 문서에 style태그를 사용하여 CSS 코드를 작성하는 방법입니다.


                <style type="text/css">
                  p {color:red;} 
                </style>
외부 스타일 시트
(External Style Sheet)

HTML 문서에 link 태그를 사용하여 CSS 확장명을 갖은 파일과 연결하고 그 CSS파일안에서 CSS 코드를 작성하는 방법입니다.


                <link rel="stylesheet" type="text/css" href="style.css">

style.css 

                p {color:yellow;}

Part2-3CSS 작성법

  • 기본작성법

    선택자 { 속성:속성값; 속성:속성값;…. 속성:속성값; }

    
              h2{display:inline-block;color:#fff;background:blue;}

    선택자 작성에 대한 자세한 사용법은 해당링크 에서 확인

  • 우선순위

    1. 명시도 점수가 높은 선언이 우선
    2. 점수가 같은 경우, 가장 마지막에 해석되는 선언이 우선
    3. 명시도는 상속 규칙보다 우선
    4. !important가 적용된 선언 방식이 다른 모든 방식보다 우선(중요도)
    
              h2{background:blue !important;}

    !important : 점수 무한대, important가 겹칠때는 그들끼리 명시도 점수를 다시 확인

    
              <h2 id="taran" class="title" style="background:yellow">타란유엑컨설팅</h2>

    인라인 선언 : 점수 1000점
